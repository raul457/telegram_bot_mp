# 🤖 Telegram Bot com Mercado Pago PIX

Bot profissional para vendas de assinaturas via Telegram com integração REAL ao Mercado Pago (PIX).

## 🚀 Funcionalidades

✅ Pagamento REAL via PIX com QR Code
✅ Confirmação automática via webhook
✅ 3 planos de assinatura (Semanal, Mensal, Anual)
✅ Expiração automática de acesso
✅ Painel administrativo completo
✅ Notificações automáticas
✅ Controle de acesso a grupos privados
